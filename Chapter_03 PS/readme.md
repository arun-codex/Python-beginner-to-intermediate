## 📝 Practice Problems

This folder contains exercises for Chapter 3. Work through these problems to master string slicing and formatting!

### 🔍 Problem Breakdown

| File | Problem Description | Key Concept Used |
| :--- | :--- | :--- |
| 🐍 **[`problem_01.py`](./problem_01.py)** | Write a program to display a user-entered name followed by "Good Afternoon" using `input()`. | String Concatenation |
| 🐍 **[`problem_02.py`](./problem_02.py)** | Write a program to fill in a letter template given below with name and date. | `replace()` method |
| 🐍 **[`problem_03.py`](./problem_03.py)** | Write a program to detect double spaces in a string. | `find()` or `count()` |
| 🐍 **[`problem_04.py`](./problem_04.py)** | Replace the double spaces from Problem 3 with single spaces. | `replace()` method |
| 🐍 **[`problem_05.py`](./problem_05.py)** | Write a program to format a given letter using escape sequence characters. | Escape Sequences (`\n`, `\t`) |

---

## 💡 How to run

To execute these practice files, open your terminal and run:

```bash
# Example
python problem_01.py
```

> [!TIP]
> String methods like `.replace()` **do not modify** the original string. They return a *new* string. So always remember to reassign it if you want to keep the changes! 
> `my_str = my_str.replace("old", "new")`
