## 🎯 Overview
This chapter introduces control flow to your Python programs. Instead of running every single line of code from top to bottom, conditionals allow your program to make decisions using `if`, `elif`, and `else` statements.

## 🔑 Learning Objectives
- Understand how boolean logic drives control flow.
- Write basic `if` statements.
- Build `if-elif-else` ladders for multiple conditions.
- Differentiate between a single `if-elif-else` ladder and multiple independent `if` statements.
- Use logical operators (`and`, `or`, `not`) inside conditions.
