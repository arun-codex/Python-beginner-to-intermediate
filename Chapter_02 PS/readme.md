## 📝 Practice Problems

This folder contains exercises designed to test your understanding of Chapter 2. Try to build the logic yourself before checking the provided solutions.

### 🔍 Problem Breakdown

*(Note: Depending on the specific problems provided in the course, the files below correspond to the exercises for Chapter 2)*

| File | Problem Description | Key Concept Used |
| :--- | :--- | :--- |
| 🐍 **`Problem_1.py`** | Write a program to add two numbers. | Variables, Arithmetic Operators |
| 🐍 **`Problem_2.py`** | Write a program to find the remainder when a number is divided by `z`. | Modulo Operator (`%`) |
| 🐍 **`Problem_3.py`** | Check the type of a variable assigned using `input()`. | `input()`, `type()` |
| 🐍 **`Problem_4.py`** | Use comparison operators to find out whether a given variable `a` is greater than `b`. | Relational/Comparison Operators |
| 🐍 **`Problem_5.py`** | Write a program to find an average of two numbers entered by the user. | `input()`, Typecasting, Arithmetic |
| 🐍 **`Problem_6.py`** | Write a program to calculate the square of a number entered by the user. | Arithmetic Operators (`**`) |

---

## 💡 How to run

To run any of the practice scripts, use your terminal:

```bash
# Example
python Problem_1.py
```

> [!TIP]
> Remember that the `input()` function always returns a **string** (`str`). If you want to do math with the user's input, you MUST typecast it to an `int` or `float` first!
> Example: `age = int(input("Enter your age: "))`
