## 📝 Practice Problems

This directory contains the solutions to the first practice set. Try to solve these problems on your own before checking the code here!

### 🔍 Problem Breakdown

| File | Problem Description | Key Concept Used |
| :--- | :--- | :--- |
| 🐍 **[`Problem_1.py`](./Problem_1.py)** | Write a program to print a multi-line poem. | Multi-line strings / `print()` |
| 🐍 **[`Problem_3.py`](./Problem_3.py)** | Write a program using an external module (`pyttsx3`) to convert text into speech. | External Modules & `pip` |
| 🐍 **[`Problem_4.py`](./Problem_4.py)** | Write a program using the built-in `os` module to list all the contents of a directory. | Built-in Modules (`os`) |

---

## 💡 How to run

To run any of the practice scripts, use the terminal and execute the file with python:

```bash
# Example for Problem 1
python Problem_1.py
```

> [!NOTE]
> For **Problem 3**, you might need to install the `pyttsx3` module first. Run this command in your terminal:
> `pip install pyttsx3`
